Privacy Policy

Last updated: November 2024

Introduction

Memory Vault is committed to protecting your privacy. This Privacy Policy explains how we handle your data when you use our Chrome extension. We believe in complete transparency and local-first data storage.

Data Collection and Storage

We do not collect or store any of your data on external servers. All your chat memories and data are stored locally in your browser using Chrome's built-in storage APIs.

The only external service we interact with is the Google Gemini API, which is used for:

  • Creating embeddings for your memories
  • Processing and extracting information from chat conversations

Google API Usage

Memory Vault requires a Google API key to function. This key:

  • Is stored locally in your browser
  • Is only used to communicate with Google's Gemini API
  • Is never transmitted to any other servers
  • Can be removed at any time through the extension settings

Data Access and Control

You have complete control over your data:

  • All memories are stored locally in your browser
  • You can view all stored memories at any time
  • You can delete individual memories or clear all data
  • Uninstalling the extension will remove all stored data

Browser Permissions

Memory Vault requires the following permissions:

  • Storage: To store your memories and settings locally
  • Active Tab: To interact with ChatGPT and Claude interfaces
  • Web Request: To detect and process chat messages

Updates to Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the "Last updated" date at the top of this page.
